📤 MilkDown 编辑器导出格式详解

🎯 支持的导出格式

✅ 当前支持的两种格式：

  • Markdown - 纯文本格式 (.md)
  • HTML - 网页格式 (.html)

📝 格式对比示例

📄 Markdown 格式

文件扩展名： .md

MIME类型： text/markdown

特点： 纯文本，易于编辑

# 标题
这是**粗体**文本

## 列表
- 项目1
- 项目2

## 代码
\`\`\`javascript
console.log("Hello");
\`\`\`

🌐 HTML 格式

文件扩展名： .html

MIME类型： text/html

特点： 可直接在浏览器显示

<h1>标题</h1>
<p>这是<strong>粗体</strong>文本</p>
<h2>列表</h2>
<ul>
<li>项目1</li>
<li>项目2</li>
</ul>
<h2>代码</h2>
<pre><code>console.log("Hello");</code></pre>

🔍 格式详细信息

1. Markdown 格式

格式类型： 纯文本 编码： UTF-8 用途： 博客、文档、笔记 优点： - 体积小，易于存储 - 版本控制友好 - 跨平台兼容 - 手动编辑简单 导出示例： ```markdown # 我的第一篇文章 ## 介绍 这是一个**Markdown**编辑器。 ## 功能 - 所见即所得 - 支持代码高亮 - 表格编辑 ```

2. HTML 格式

格式类型： 标记语言 编码： UTF-8 用途： 网页、邮件、模板 优点： - 直接在浏览器显示 - 支持CSS样式 - 富文本格式 - 交互功能 导出示例： ```html

我的第一篇文章

介绍

这是一个Markdown编辑器。

功能

  • 所见即所得
  • 支持代码高亮
  • 表格编辑
```

❌ 不支持的格式

目前不支持以下格式：

  • ❌ JSON - 不是JSON格式
  • ❌ XML - 不是XML格式
  • ❌ DOCX - 不是Word文档
  • ❌ PDF - 不是PDF格式（除非额外添加）
  • ❌ 纯文本 - 不是普通的txt格式
